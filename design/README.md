# 菜园管家视觉设计稿

设计稿入口：

```text
design/caicai-visual-design-board.html
```

这版设计稿用于统一 Android 后续改造的视觉方向，重点对齐参考图里的微缩菜园风格：

- 阳光、草地、木质菜床、石板路、温室和菜园工具感。
- 菜地地图从列表式管理转为单块大地块展示，通过地块切换查看其他菜地。
- 菜地地图采用高保真等距菜园场景资产作为底图，而不是用简单圆点或 CSS 方块模拟。
- 作物、温室、木屋、石板路、灌溉线和木牌直接体现在地图底图里，交互层只负责地块切换、任务标签和状态浮层。
- 日历用石板路时间线，记录页用菜园手账/照片卡片。
- 添加种植、记录操作、添加地块用半页表单/木牌式卡片承接，不再像普通后台表单。
- 最新完善版把记录相册、添加种植、记录操作、添加地块也接入真实菜园局部图，统一照片感、玻璃浮层、木牌卡片和石板路时间线。
- 今日页和地图页共用同一张菜园资产，确保从工作台进入地图时视觉连续。
