# 菜园管家 1.2.0

本版本重点完善真实种植记录、作物逐日生长和菜园布局体验。

## 新增与改进

- 种菜时必须明确选择种子、种苗或插条，并记录播种、移栽或扦插日期。
- 生长阶段、任务提醒和预计采收时间会结合种植方式与每种作物自己的周期计算。
- 15 种内置作物分别配置四阶段图片、阶段天数、冠幅区间、株高和增长曲线，不再共用统一缩放规则。
- 成熟番茄、黄瓜和茄子的冠幅提高到约 1.0-1.25 个格子；第 104 天茄子会显示为挂果采收期。
- 空心菜和韭菜新增独立的幼苗、成长、成熟和采收素材。
- 收紧 8 x 8 田块之间的视觉间距，并移除普通点选时含义不清的木质框。
- 增加远小近大的透视景深、逐格命中和前后遮挡；每格只绘制草地或土壤，修复放大后像方块堆叠的问题。
- 拖拽到已占用田块时改为交换两个对象，不再覆盖并丢失目标对象。

## 验证

- 14 个单元测试全部通过。
- Debug 构建与 Android Lint 通过。
- 已在 Android 17、1080 x 2400 模拟器完成种植方式、逐日成长、田块交换和主要菜园交互验收。

如果设备上安装的是 Debug 开发包，需要先卸载后再安装本正式版。已经安装正式版 1.1.0 的设备可以直接覆盖升级。
