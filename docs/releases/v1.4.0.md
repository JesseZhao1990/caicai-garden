# 菜园管家 1.4.0

本版本重点改善作物与土地的接触关系，让不同株型真正“长在土里”，减少插入感、贴纸感和漂浮感。

## 新增与改进

- 为 15 种作物分别配置四个生长阶段的根部锚点、接触宽度和入土深度，不再统一使用素材最底部定位。
- 新增直立茎秆、莲座叶菜、匍匐藤蔓、丛生菜、鳞茎和地下根菜六种落地方式，阴影与土面遮挡会随株型变化。
- 作物旋转和风吹摆动改为围绕根部锚点进行，避免整棵作物像贴纸一样悬空晃动。
- 接触阴影改为按素材有效内容绘制，修复透明留白导致阴影几乎不可见的问题。
- 在根部增加浅种植穴和不连续的小土块遮挡，使茎秆、叶柄与泥土自然衔接，同时避免形成横向底座。
- 胡萝卜、萝卜等根菜在成熟阶段以根冠对齐土面并隐藏地下部分；黄瓜、西葫芦等藤蔓继续保留贴地果实和卷须。
- 拖拽预览的阴影同样按有效内容缩放，移动作物时不再出现过淡或尺寸失真的悬浮阴影。

## 验证

- 15 个单元测试全部通过，新增关键株型根部锚点与接触策略覆盖。
- Debug 构建与 Android Lint 通过。
- 已在 Android 17、1080 x 2400 模拟器完成查看态、移动态和两级放大画面验收。
- 直立作物、叶菜和藤蔓的根部接触、阴影、土面遮挡及摆动支点均已通过设备截图复核。

如果设备上安装的是 Debug 开发包，需要先卸载后再安装本正式版。已经安装正式版 1.1.0、1.2.0 或 1.3.0 的设备可以直接覆盖升级。
