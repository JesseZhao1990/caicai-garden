# 菜园管家 1.6.1

本版本优化应用内升级体验，并让安装包下载真正支持后台执行。

## 新增与改进

- 安装包改由 Android 系统下载服务执行，切换页面或把应用退到后台后仍会继续下载。
- 保存下载任务、版本和文件信息；应用进程被回收后，重新打开可以恢复进度或安装状态。
- 下载完成后继续执行 SHA-256 完整性校验，校验通过后才允许进入安装流程。
- 下载状态只保留一个清晰的状态标签，移除重复且不可点击的“下载中”按钮。
- 使用与菜园主题一致的自定义圆角进度条，移除系统进度条末尾突兀的圆点。
- 增加“可离开此页面”和“后台下载已开启”等提示，明确告知用户后台下载能力。
- 安装完成或当前版本已是目标版本时，自动清理遗留下载任务和安装包。

## 验证

- Debug 单元测试、Android Lint 和 Debug APK 构建全部通过。
- 在 Android 模拟器触发真实 Release APK 下载，并在下载中强制停止应用。
- 强制停止前系统任务已下载约 56%，重新打开后任务自动恢复并完成到 100%。
- 下载完成后 SHA-256 校验通过，页面正确进入“等待安装”状态。
- 已确认下载态不再显示重复按钮，进度条末尾不再出现额外圆点。

已经安装正式版 1.1.0 至 1.6.0 的设备可以直接覆盖升级。
