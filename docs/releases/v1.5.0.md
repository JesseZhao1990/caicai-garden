# 菜园管家 1.5.0

本版本完成菜园核心操作与作物养护体系升级：种植、浇水、施肥和采摘都融合到菜园地图，并将作物库从 15 种扩展到 32 种。

## 新增与改进

- 菜园地块切换默认折叠，展开后集中管理地块并可直接添加新地块。
- 种植入口融合到地图空闲菜格；选择作物、种植方式、日期和数量后直接落到目标位置。
- 种植面板新增作物分类筛选，32 种作物可按果菜、瓜类、叶菜、香草、葱蒜、根茎、豆类等类别快速查找。
- 作物种植方式按品种动态变化，新增鳞茎/蒜瓣定植和种薯/块茎种植等方式。
- 浇水、施肥和采摘改为选中地图作物后的快捷操作，并自动生成菜园足迹。
- 浇水周期综合作物需水、地块类型、生长阶段、温度和降雨动态计算，不再依赖阶段中文名称判断。
- 32 种作物均配置四段连续生长周期、施肥节点、采摘窗口与独立采摘间隔。
- 连续采收作物新增周期追肥；部分采摘后可继续保留作物并按作物间隔再次提醒。
- 采收期结束后停止重复养护提醒，并通过二次确认安全结束本茬、清空菜格，同时保留历史记录。
- 生长进度、施肥和采摘提醒统一使用有效生长日，基础温度和适宜温区正式参与计算。
- 底部导航精简为今日、菜园、日历和我的；操作记录自动归入“我的”菜园足迹。
- 优化版本升级卡片、地块切换、地图缩放与布局控制，使整体风格保持一致。

## 验证

- Debug 单元测试全部通过，覆盖作物视觉映射、动态种植方式、独立采摘间隔、部分采摘、连续追肥、温度生长和生命周期结束。
- Android Lint、Debug APK 构建及差异检查通过。
- 32 套作物素材均存在对应业务配置和视觉映射。
- 已在 Android 模拟器完成种植分类、日期选择、特殊种植方式、地图养护和生命周期确认交互回归。

如果设备上安装的是 Debug 开发包，需要先卸载后再安装本正式版。已经安装正式版 1.1.0 至 1.4.0 的设备可以直接覆盖升级。
