# 菜园管家 1.6.0

本版本将菜园升级为沉浸式全屏地图，并补齐地块删除、旧装饰清理和底部导航稳定性。

## 新增与改进

- 菜园地图改为全屏主界面，地图延伸到页面底部，减少卡片和固定信息区对画面的占用。
- 地块入口收敛为左上角悬浮按钮，默认只显示当前地块和数量，点击后再展开完整地块管理。
- 养护/布局模式改为独立悬浮折叠按钮，与地块入口保持同一行，不再额外占据地图高度。
- 地块管理新增删除能力，每个地块都可单独删除。
- 删除前增加二次确认，并明确提示地块中仍在生长的作物数量。
- 删除地块后停止相关养护提醒、移除地图布局，同时继续保留历史菜园足迹。
- 移除菜地中的小木牌；新建布局不再生成，已有用户存档中的旧木牌也会在读取和保存时自动过滤。
- 底部导航统一使用固定定位，解决进入菜园时整组菜单发生纵向位移的问题。
- 缩放、养护操作、布局重置和底部导航继续以悬浮方式覆盖在地图上，不影响菜园全屏展示。

## 验证

- Debug 单元测试全部通过，新增旧木牌过滤和地块删除数据一致性测试。
- Android Lint、Debug APK 构建和差异检查通过。
- 已在 Android 模拟器完成全屏地图、按钮式地块入口、地块切换、删除确认和删除后自动切换回归。
- 已对比“今日”和“菜园”页面的底部菜单坐标，切换前后位置完全一致。

如果设备上安装的是 Debug 开发包，需要先卸载后再安装本正式版。已经安装正式版 1.1.0 至 1.5.0 的设备可以直接覆盖升级。
