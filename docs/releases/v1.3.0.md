# 菜园管家 1.3.0

本版本重点改善菜地与花园背景的融合、土地分格质感，以及成熟作物的显示完整性。

## 新增与改进

- 移除由立体草块组成的整片悬浮草皮，菜地外围改为自然融入花园背景的连续耕作土地轮廓。
- 保留平面等距土地方格，格子之间使用窄而连续的深色泥土小沟，不再出现草色缝隙或厚侧壁。
- 细化土壤纹理颗粒，通过连续重复采样保持整片菜地材质一致，并避免明显拼接。
- 查看模式使用较柔和的格子边缘，移动、清空和种植等编辑状态适度增强定位反馈。
- 作物根部与土壤接触位置重新校准，同时保留远小近大、前后遮挡和透视命中逻辑。
- 修复成熟白菜原图顶部被裁切的问题，收获期现在显示轮廓完整的单株白菜。
- 移除收获期作物右上角含义不清的绿色圆形徽标；成熟信息继续在作物详情中展示。

## 验证

- 14 个单元测试全部通过。
- Debug 构建与 Android Lint 通过。
- 已在 Android 17、1080 x 2400 模拟器完成默认视图、两级放大、查看态和移动态画面验收。
- 白菜完整显示、成熟圆标移除、平面土地方格和泥土沟均已通过设备截图复核。

如果设备上安装的是 Debug 开发包，需要先卸载后再安装本正式版。已经安装正式版 1.1.0 或 1.2.0 的设备可以直接覆盖升级。
